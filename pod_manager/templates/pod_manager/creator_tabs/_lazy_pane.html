{% comment %} Reusable lazy tab-pane, split into two nested elements with distinct jobs: • The OUTER .tab-pane carries the #boosted-region swap target/select/swap, so forms and links inside the lazily-loaded body boost against #boosted-region — a clean region swap that preserves the out-of-region floating player. It fires no request itself; it is purely the inheritance source for the loaded content. (Boost itself is inherited from the app-wide hx-boost wrapper.) • The INNER loader does the one-shot GET of the tab body into the pane. Its self-targeting attrs (hx-target = this pane, innerHTML, select=unset) live on this SEPARATE child and are marked hx-disinherit="*" so they never cascade to the loaded content. The previous single-div version put those on the pane and used hx-disinherit to hide them — but disinherit blocks inheritance entirely, so boosted forms in the body inherited NO target and fell back to htmx's boosted default (), whose innerHTML swap nuked the floating player. Params: tab_name, pane_id, link_id, active_tab (= request.GET.tab). {% endcomment %}
