Keeping it here holds the history element stable at #boosted- region across swaps. The hx-boost / hx-target / hx-select / hx-swap / hx-push-url attributes live on an OUTER wrapper in base.html (enclosing the nav + this region), NOT on the region itself — so nav links, which sit outside the region, are still boosted (a full reload would destroy the out-of-region floating player). Boosted links inside the swapped-in region inherit those attributes from that wrapper. {% endcomment %}
